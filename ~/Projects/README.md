# Projects Directory

This directory contains personal projects organized by purpose.

## Structure

- **experiments/** - Quick tests, proof-of-concepts, and experimental code
- **personal/** - Personal projects and side projects
- **learning/** - Tutorials, courses, and educational projects
- **playground/** - Sandbox for trying out new technologies and frameworks

## Usage Tips

- Keep projects self-contained in their own subdirectories
- Add a README to each project explaining what it is
- Use descriptive names for project folders
- Archive or delete projects you're no longer working on
